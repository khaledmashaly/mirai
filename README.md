# Mirai

A project to explore:

- Kotlin
- Spring webflux
- Gradle
- GraphQL

among other technologies.

## Common Tasks

- Run the project
  1. Run Docker compose
      ```bash
      docker-compose up -d
      ```
  2. Run Spring
      ```bash
      ./gradlew bootRun
      ```
- Build the project
    ```bash
    ./gradlew build
    ```
